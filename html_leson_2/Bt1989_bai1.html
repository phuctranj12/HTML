WritePad Editor

WritePad is an editor appliction that allows you to creat WEb pages easily and quickli.
Steps to open WritePad application :

  1. Click Start

Advantages of using a Writepad editor :

  • It allows faster creation of Web pages.
  • It allows easier updation of text or images in WritePad edittor.
